Introduction to Ansible

Ansible is an open-source automation tool used for configuration management, application deployment, and infrastructure provisioning. It works in an agentless architecture using SSH to communicate with managed nodes. Automation tasks are written in YAML-based playbooks, which makes them simple and readable. Ansible helps automate repetitive tasks like package installation, service management, and server configuration across multiple systems.

Step 1 — Install Ansible

On your control machine:

Ubuntu
apt install ansible -y

RHEL / CentOS
yum install ansible -y

Check version:
ansible --version

Step 2 — Setup Passwordless SSH

Ansible connects using SSH.

Create SSH key:
ssh-keygen

Copy key to server:
ssh-copy-id root@server1

Now you can login without password.

Step 3 — Test Connection

Run:
ansible all -m ping

Output:
server1 | SUCCESS
server2 | SUCCESS

Means Ansible can reach the servers.
