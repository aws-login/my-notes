What is YAML?

Simple Definition

YAML (YAML Ain’t Markup Language) is a human-readable data format used to write configuration files.

In Ansible, YAML is used to write playbooks and automation tasks.

It is designed to be simple, easy to read, and easy to write.

Why Ansible Uses YAML

YAML is used in Ansible because:

  • It is very readable
  • Uses simple structure
  • Easy to understand automation tasks
  • No complex syntax
